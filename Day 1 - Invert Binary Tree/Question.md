# Invert Binary Tree

- Link: https://leetcode.com/explore/challenge/card/june-leetcoding-challenge/539/week-1-june-1st-june-7th/3347/

## Invert a binary tree.

### Example:

#### Input:

```java
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```


#### Output:

```java
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
